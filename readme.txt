New Year Countdown

== Description ==

A nice countdown to new year widget. An example is available here - http://newyear0.com/

bugs? ideas? email me: wb@christmas0.com

== Installation ==

1. Upload `new-year-countdown` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add New Year Countdown widget (in the 'Widgets' menu) to any sidebar.
4. Set widget width (default is 200 pixels).
